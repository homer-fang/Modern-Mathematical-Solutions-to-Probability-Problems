# P1654 OSU!：用条件期望理解期望 DP

## 题目描述

有 $n$ 次相互独立的操作。第 $i$ 次操作以概率 $p_i$ 成功，记为 $1$；以概率 $1-p_i$ 失败，记为 $0$。最终会得到一个长度为 $n$ 的 01 串。

计分规则如下：对每一段极长连续的 $1$，如果长度为 $x$，则这一段贡献 $x^3$ 分。

要求计算总分的期望，结果四舍五入保留 $1$ 位小数。

## 输入格式

第一行一个正整数 $n$。接下来 $n$ 行，每行一个 $[0,1]$ 之间的实数 $p_i$，表示第 $i$ 次操作成功的概率。

## 输出格式

输出一个实数，表示期望总分，保留 $1$ 位小数。

## 输入输出样例

### 输入

```text
3
0.5
0.5
0.5
```

### 输出

```text
6.0
```

## 样例解释

长度为 $3$ 的所有 01 串共有 $8$ 个：

```text
000, 001, 010, 011, 100, 101, 110, 111
```

对应得分分别为：

```text
0, 1, 1, 8, 1, 2, 8, 27
```

总和为 $48$。因为每个串出现概率都是 $1/8$，所以期望为 $48/8 = 6.0$。

数据范围：$n \le 10^5$。

 

# 一、题目与概率空间

这道题表面上是在求一个 01 串得分的期望，本质上是在问：每一次新操作发生以后，总分的期望会增加多少。

为了把“随机的 01 串”说清楚，我们先把所有可能结果放进同一个概率空间里。

长度为 $n$ 的所有 01 串构成样本空间：

$$\Omega = \{0,1\}^n$$

一个样本点记为：

$$\omega = (\omega_1, \omega_2, \ldots, \omega_n)$$

其中 $\omega_i = 1$ 表示第 $i$ 次操作成功；$\omega_i = 0$ 表示第 $i$ 次操作失败。

因为 $\Omega$ 是有限集合，所以可以直接取全体子集作为 $\sigma$-代数：

$$\mathcal{F} = 2^{\Omega}$$

定义概率测度：

$$\mathbb{P}(\{\omega\}) = \prod_{i=1}^{n} p_i^{\omega_i}(1-p_i)^{1-\omega_i}$$

这就是每个具体 01 串出现的概率。

也就是说，如果某个位置是 $1$，就乘上这一位成功的概率 $p_i$；如果某个位置是 $0$，就乘上失败的概率 $1-p_i$。整串出现的概率就是这些概率的乘积。

定义坐标随机变量：

$$X_i(\omega) = \omega_i$$

于是：

$$\mathbb{P}(X_i = 1) = p_i, \quad \mathbb{P}(X_i = 0) = 1 - p_i$$

并且 $X_1, X_2, \ldots, X_n$ 相互独立。

 

# 二、过滤族与后缀长度

## 1. 引入过滤族

过滤族听起来很抽象，其实在这道题里可以先把它理解成“到某一步为止，我们已经看见了哪些结果”。

设：

$$\mathcal{F}_i = \sigma(X_1, X_2, \ldots, X_i)$$

它表示由前 $i$ 次操作结果所能区分出来的全部事件。

直观地说，$\mathcal{F}_i$ 表示：到第 $i$ 步为止，我们已经掌握的全部信息。

例如：

$$\mathcal{F}_3 = \sigma(X_1, X_2, X_3)$$

它可以表达下面这些事件：

$$\{X_1 = 1\}, \quad \{X_2 = 0, X_3 = 1\}, \quad \{X_1 = 1, X_2 = 1, X_3 = 0\}, \quad \{X_1 + X_2 + X_3 \ge 2\}$$

这些事件只依赖前 $3$ 次操作结果，所以都属于 $\mathcal{F}_3$。

但是 $\{X_5 = 1\}$ 不属于 $\mathcal{F}_3$，因为只知道前三次操作结果时，还无法判断第五次操作是否成功。

过滤族是一串不断变细、不断增大的信息结构：

$$\mathcal{F}_0 \subseteq \mathcal{F}_1 \subseteq \mathcal{F}_2 \subseteq \cdots \subseteq \mathcal{F}_n$$

第 $0$ 步时几乎什么都不知道；第 $1$ 步时知道了 $X_1$；第 $2$ 步时知道了 $X_1, X_2$；第 $i$ 步时知道了 $X_1, \ldots, X_i$。

在这道题中，引入过滤族的核心作用是说明：$L_{i-1}$ 只由前 $i-1$ 次操作结果决定，因此它是 $\mathcal{F}_{i-1}$-可测的，也就是“过去已经知道的量”。

而 $X_i$ 是第 $i$ 次操作的结果，它与过去信息 $\mathcal{F}_{i-1}$ 独立，所以：

$$\mathbb{E}[X_i \mid \mathcal{F}_{i-1}] = p_i$$

即使已经知道前 $i-1$ 次操作的全部结果，第 $i$ 次操作成功的条件期望仍然是 $p_i$。

这句话是后面递推的关键：过去的量可以看成已经定下来了，而第 $i$ 次是否成功仍然只按自己的概率 $p_i$ 来平均。

 

## 2. 定义连续成功后缀长度

设 $L_i$ 表示前 $i$ 次操作中，以第 $i$ 位结尾的连续成功后缀长度。

例如 `1 1 0 1 1 1`，此时 $L_6 = 3$。

严格地说，$L_i$ 满足递推关系：

$$L_i = X_i(L_{i-1} + 1), \quad L_0 = 0$$

若 $X_i = 1$，则 $L_i = L_{i-1} + 1$；若 $X_i = 0$，则 $L_i = 0$。

所以 $L_i$ 只关心“当前末尾这一段连续的 $1$ 有多长”，不关心前面已经断开的那些段。

 

## 3. 定义前缀总分随机变量

设 $S_i$ 表示前 $i$ 次操作形成的 01 串的总分。题目最后要求的答案就是 $\mathbb{E}[S_n]$。

接下来不直接计算 $S_i$ 的完整分布，而是只看从 $S_{i-1}$ 到 $S_i$ 多出来了多少分。

若第 $i$ 次失败（$X_i = 0$），分数不变：$S_i - S_{i-1} = 0$。

若第 $i$ 次成功（$X_i = 1$），末尾连续成功段被延长，长度从 $L_{i-1}$ 变为 $L_{i-1} + 1$，这一段的贡献从 $L_{i-1}^3$ 变为 $(L_{i-1} + 1)^3$。

于是对每个样本点 $\omega$，都有如下恒等式：

$$S_i - S_{i-1} = X_i\left((L_{i-1} + 1)^3 - L_{i-1}^3\right)$$

这里前面的 $X_i$ 是一个开关：如果第 $i$ 次失败，$X_i=0$，新增分数就是 $0$；如果成功，$X_i=1$，才会把末尾这一段的新增贡献算进去。

展开 $(L + 1)^3 - L^3 = 3L^2 + 3L + 1$，所以：

$$S_i - S_{i-1} = X_i(3L_{i-1}^2 + 3L_{i-1} + 1)$$

# 三、条件期望推出递推

## 1. 答案递推

现在真正要做的，就是对上面的新增分数取期望。

因为 $L_{i-1}$ 只由前 $i-1$ 次操作决定，所以 $L_{i-1}$ 以及 $3L_{i-1}^2 + 3L_{i-1} + 1$ 都是 $\mathcal{F}_{i-1}$-可测的。

又因为 $X_i$ 与 $\mathcal{F}_{i-1}$ 独立，所以 $\mathbb{E}[X_i \mid \mathcal{F}_{i-1}] = p_i$。

直观地说，$3L_{i-1}^2 + 3L_{i-1} + 1$ 是“过去已经决定的系数”，而 $X_i$ 是“当前这一步还没发生的随机开关”。条件期望做的事，就是先固定过去，再把这个随机开关平均掉。

于是：

$$
\begin{aligned}
\mathbb{E}[S_i - S_{i-1}]
&= \mathbb{E}\left[X_i(3L_{i-1}^2 + 3L_{i-1} + 1)\right] \\
&= \mathbb{E}\left[\mathbb{E}\left[X_i(3L_{i-1}^2 + 3L_{i-1} + 1) \mid \mathcal{F}_{i-1}\right]\right] \\
&= \mathbb{E}\left[(3L_{i-1}^2 + 3L_{i-1} + 1)\,\mathbb{E}[X_i \mid \mathcal{F}_{i-1}]\right] \\
&= \mathbb{E}\left[(3L_{i-1}^2 + 3L_{i-1} + 1)\,p_i\right] \\
&= p_i\,\mathbb{E}[3L_{i-1}^2 + 3L_{i-1} + 1] \\
&= p_i\left(3\mathbb{E}[L_{i-1}^2] + 3\mathbb{E}[L_{i-1}] + 1\right)
\end{aligned}
$$

设 $a_i = \mathbb{E}[L_i]$，$b_i = \mathbb{E}[L_i^2]$，$\mathrm{ans}_i = \mathbb{E}[S_i]$，则：

$$\mathrm{ans}_i = \mathrm{ans}_{i-1} + p_i(3b_{i-1} + 3a_{i-1} + 1)$$

所以答案递推只需要知道两件事：上一轮末尾连续成功长度的一阶矩 $a_{i-1}$，以及二阶矩 $b_{i-1}$。

 

## 2. 推出 $a_i$ 和 $b_i$ 的递推

既然答案需要 $a_i$ 和 $b_i$，我们还要顺手维护它们。

由 $L_i = X_i(L_{i-1} + 1)$：

$$
\begin{aligned}
a_i
&= \mathbb{E}[L_i] \\
&= \mathbb{E}[X_i(L_{i-1} + 1)] \\
&= p_i(a_{i-1} + 1)
\end{aligned}
$$

这一步同样使用了 $X_i$ 与 $\mathcal{F}_{i-1}$ 独立，以及 $L_{i-1}+1$ 是 $\mathcal{F}_{i-1}$-可测的。

用普通 DP 的语言说，就是：如果第 $i$ 次成功，后缀长度在原来基础上加一；如果失败，后缀长度归零。再乘上成功概率 $p_i$，就得到期望后缀长度。

对于二次矩，$L_i^2 = X_i^2(L_{i-1} + 1)^2$。由于 $X_i \in \{0,1\}$，有 $X_i^2 = X_i$，于是：

二阶矩也是同样的思路，只不过把后缀长度平方以后再取期望。

$$
\begin{aligned}
b_i &= \mathbb{E}[L_i^2] = \mathbb{E}[X_i(L_{i-1} + 1)^2] \\
&= p_i\,\mathbb{E}[L_{i-1}^2 + 2L_{i-1} + 1] \\
&= p_i(b_{i-1} + 2a_{i-1} + 1)
\end{aligned}
$$

 

## 3. 最终递推式

初始值 $a_0 = b_0 = \mathrm{ans}_0 = 0$。对于 $i = 1, 2, \ldots, n$：

$$
\begin{aligned}
\mathrm{ans}_i &= \mathrm{ans}_{i-1} + p_i(3b_{i-1} + 3a_{i-1} + 1) \\
a_i &= p_i(a_{i-1} + 1) \\
b_i &= p_i(b_{i-1} + 2a_{i-1} + 1)
\end{aligned}
$$

$\mathrm{ans}_n$ 就是答案。

这就是整道题的算法核心：从左到右扫一遍，每读入一个 $p_i$，更新当前答案、一阶矩和二阶矩。

# 四、为什么不需要维护三次矩

不需要维护三次矩，核心原因在于：

每一步真正加入答案的，不是新的整段贡献 $(L_{i-1}+1)^3$，而是旧贡献被替换以后多出来的部分：

$$(L_{i-1}+1)^3-L_{i-1}^3$$

这个差值展开以后，只会含有二次项、一次项和常数项。

这点很容易看错。题目里的得分确实是三次方，但我们每一步加到答案里的不是“整段三次方”，而是“这一位成功以后，末尾这段比原来多了多少”。

设第 $i-1$ 位结尾的连续成功长度是 $L_{i-1}$。

如果第 $i$ 次成功，那么末尾这一段长度从 $L_{i-1}$ 变成 $L_{i-1}+1$。

原来这一段对总分的贡献是：

$$L_{i-1}^3$$

现在这一段对总分的贡献是：

$$(L_{i-1}+1)^3$$

所以真正新增的分数是：

$$(L_{i-1}+1)^3-L_{i-1}^3$$

展开：

$$
(L+1)^3-L^3
=L^3+3L^2+3L+1-L^3
=3L^2+3L+1
$$

三次项 $L^3$ 被抵消了。

因此答案增量只需要 $\mathbb{E}[L_{i-1}^2]$ 和 $\mathbb{E}[L_{i-1}]$，不需要 $\mathbb{E}[L_{i-1}^3]$。

比如当前末尾连续成功长度是 $2$：

原来贡献是：$2^3 = 8$

如果下一位成功，长度变成 $3$，新贡献是 $3^3 = 27$。

但是新增分数不是 $27$，而是 $27-8=19$。

而：

$$3 \cdot 2^2 + 3 \cdot 2 + 1 = 12 + 6 + 1 = 19$$


这就是为什么只需要维护一阶矩和二阶矩。

如果额外维护三次矩 $c_i = \mathbb{E}[L_i^3]$。

它确实也有递推：

$$c_i = p_i(c_{i-1} + 3b_{i-1} + 3a_{i-1} + 1)$$

但是 $c_i$ 表示的是：

第 $i$ 位结尾的连续成功后缀长度的三次方期望。

它不是前 $i$ 位的总分期望。

总分期望是 $\mathrm{ans}_i = \mathbb{E}[S_i]$，而不是 $\mathbb{E}[L_i^3]$。

所以如果把 $c_i$ 看成答案，就只统计了「当前末尾这一段」，而没有统计所有已经结束或正在延长的连续段。

换句话说，$c_i$ 不是错的量，只是不是题目要的量。题目要的是所有连续段的总分期望，而 $c_i$ 只盯着当前后缀这一段。

# 五、C++ 实现代码

代码里用 `E1` 表示 $a_i$，用 `E2` 表示 $b_i$，用 `ans` 表示 $\mathrm{ans}_i$。数组写法更贴近上面的递推式，实际也可以改成滚动变量。

```cpp
#include <bits/stdc++.h>
using namespace std;

const int MAXN = 1e5 + 10;

double E1[MAXN];
double E2[MAXN];
double ans[MAXN];

signed main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    int n;
    cin >> n;

    E1[0] = E2[0] = ans[0] = 0.0;

    for (int i = 1; i <= n; i++) {
        double p;
        cin >> p;

        ans[i] = ans[i - 1] + p * (3 * E2[i - 1] + 3 * E1[i - 1] + 1);

        E1[i] = p * (E1[i - 1] + 1);

        E2[i] = p * (E2[i - 1] + 2 * E1[i - 1] + 1);
    }
    cout << fixed << setprecision(1) << ans[n] << "\n";

    return 0;
}
```

时间复杂度为 $O(n)$，空间复杂度为 $O(n)$。由于递推只依赖上一项，也可以用滚动变量把空间优化到 $O(1)$。

# 附录一、随机变量生成的 $\sigma$-代数

下面的附录用于补足前面用到的概率论语言。第一次阅读时，可以先抓住直观含义：$\sigma$-代数描述“能分辨哪些事件”，条件期望描述“在当前信息下做平均”。

| 符号 | 含义 |
| :-- | :-- |
| $\Omega$ | 样本空间 |
| $\omega$ | 样本点 |
| $\mathcal{F}$ | $\sigma$-代数 |
| $\mathbb{P}$ | 概率测度 |
| $\mathcal{G} \subseteq \mathcal{F}$ | 子 $\sigma$-代数，表示当前掌握的信息 |
| $X \colon \Omega \to \mathbb{R}$ | 随机变量 |

由 $X$ 生成的 $\sigma$-代数记作 $\sigma(X)$：

$$\sigma(X) = \{X^{-1}(B) : B \in \mathcal{B}(\mathbb{R})\}$$

其中 $\mathcal{B}(\mathbb{R})$ 是实数轴上的 Borel $\sigma$-代数，而

$$X^{-1}(B) = \{\omega \in \Omega : X(\omega) \in B\}$$

表示事件「$X$ 的取值落在 $B$ 中」。

更完整地：

$$\sigma(X) = \{\{\omega \in \Omega : X(\omega) \in B\} : B \in \mathcal{B}(\mathbb{R})\}$$

直观理解：$\sigma(X)$ 就是只观察 $X$ 的值时，能够判断出来的所有事件。

例如 $\Omega = \{HH, HT, TH, TT\}$，设 $X$ 表示第一次抛掷结果（$H$ 记为 $1$，$T$ 记为 $0$）。只观察 $X$ 时，只能区分 $\{HH, HT\}$ 与 $\{TH, TT\}$，因此：

$$\sigma(X) = \{\varnothing, \Omega, \{HH, HT\}, \{TH, TT\}\}$$

# 附录二、条件期望

条件期望可以理解成：在只知道 $\mathcal{G}$ 这部分信息时，对随机变量 $X$ 做出的最好平均估计。

设 $(\Omega, \mathcal{F}, \mathbb{P})$ 是概率空间，$\mathcal{G} \subseteq \mathcal{F}$ 是子 $\sigma$-代数，$X$ 是非负或可积随机变量。

$\mathbb{E}[X \mid \mathcal{G}]$ 是满足下面两个条件的随机变量。

## 1. 可测性

$\mathbb{E}[X \mid \mathcal{G}]$ 是 $\mathcal{G}$-可测的：$\mathbb{E}[X \mid \mathcal{G}](\omega)$ 只依赖 $\mathcal{G}$ 中已经包含的信息。

若 $\mathcal{G}$ 分不清 $HHH$ 与 $HHT$，则必须有：

$$\mathbb{E}[X \mid \mathcal{G}](HHH) = \mathbb{E}[X \mid \mathcal{G}](HHT)$$

## 2. 部分平均性质

对任意 $A \in \mathcal{G}$，都有：

$$\int_A \mathbb{E}[X \mid \mathcal{G}](\omega)\, d\mathbb{P}(\omega) = \int_A X(\omega)\, d\mathbb{P}(\omega)$$

即：

$$\boxed{\int_{\omega \in A} \mathbb{E}[X \mid \mathcal{G}](\omega)\, \mathbb{P}(d\omega) = \int_{\omega \in A} X(\omega)\, \mathbb{P}(d\omega), \quad \forall A \in \mathcal{G}}$$

# 附录三、条件期望的有限样本空间例子

抛三次硬币，$\Omega = \{HHH, HHT, HTH, HTT, THH, THT, TTH, TTT\}$。设 $\mathcal{G} = \mathcal{F}_2$，也就是只知道前两次结果。

$\mathcal{F}_2$ 会把样本空间分成四块：$A_{HH}, A_{HT}, A_{TH}, A_{TT}$。

在 $\mathcal{F}_2$ 看来 $HHH$ 与 $HHT$ 不可区分，所以：

$$\mathbb{E}[X \mid \mathcal{F}_2](HHH) = \mathbb{E}[X \mid \mathcal{F}_2](HHT)$$

在 $A_{HH} = \{HHH, HHT\}$ 上，由部分平均性质：

$$\sum_{\omega \in A_{HH}} \mathbb{E}[X \mid \mathcal{F}_2](\omega)\,\mathbb{P}(\{\omega\}) = \sum_{\omega \in A_{HH}} X(\omega)\,\mathbb{P}(\{\omega\})$$

设该块上的常数为 $c_{HH}$，则：

$$c_{HH} = \frac{X(HHH)\,\mathbb{P}(\{HHH\}) + X(HHT)\,\mathbb{P}(\{HHT\})}{\mathbb{P}(\{HHH\}) + \mathbb{P}(\{HHT\})}$$

即：

$$\boxed{\mathbb{E}[X \mid \mathcal{F}_2](HHH) = \mathbb{E}[X \mid \mathcal{F}_2](HHT) = \frac{X(HHH)\,\mathbb{P}(\{HHH\}) + X(HHT)\,\mathbb{P}(\{HHT\})}{\mathbb{P}(\{HHH\}) + \mathbb{P}(\{HHT\})}}$$

更一般地，若 $\mathcal{G}$ 把有限样本空间分成信息块 $C_1,C_2,\ldots,C_m$，则在 $\mathbb{P}(C_j)>0$ 时：

$$
\mathbb{E}[X\mid C_j]
=
\frac{
\sum_{\omega\in C_j}X(\omega)\mathbb{P}(\{\omega\})
}{
\mathbb{P}(C_j)
}
$$

若 $\mathbb{P}(C_j)=0$，该块上的取值不影响条件期望的几乎处处意义，因此可以任意定义。

# 附录四、核心理解

条件期望不要求 $\mathbb{E}[X \mid \mathcal{G}](\omega)=X(\omega)$，而是要求：在 $\mathcal{G}$ 能分辨的每一个信息块上，把 $X$ 做加权平均。

$$\boxed{\mathbb{E}[X \mid \mathcal{G}] = \text{在只知道 }\mathcal{G}\text{ 信息时，对 }X\text{ 做出的分块平均估计}}$$

回到 P1654，最关键的一步是：

$$S_i - S_{i-1} = X_i(3L_{i-1}^2 + 3L_{i-1} + 1)$$

其中 $L_{i-1}$ 是 $\mathcal{F}_{i-1}$-可测的过去信息，$X_i$ 是与过去独立的新随机结果。因此：

$$\mathbb{E}\left[X_i(3L_{i-1}^2 + 3L_{i-1} + 1) \mid \mathcal{F}_{i-1}\right] = (3L_{i-1}^2 + 3L_{i-1} + 1)\,p_i$$

从算法角度看，它变成了三个简单递推；从概率论角度看，它是用过滤族描述「过去信息」，用条件期望处理「当前独立随机变量」，最后得到期望 DP。

# 附录五、塔式定理

塔式定理说的是：如果已经按更细的信息平均过一次，再按更粗的信息平均一次，结果等于一开始就按更粗的信息平均。

## 基本形式

先证明塔式定理的一个常用推论：

$$\mathbb{E}[\mathbb{E}[X\mid \mathcal{G}]] = \mathbb{E}[X]$$

设：

$$Z=\mathbb{E}[X\mid \mathcal{G}]$$

由条件期望的定义，$Z$ 满足两个条件：

第一，$Z$ 是 $\mathcal{G}$-可测的。

第二，对任意 $A\in\mathcal{G}$，都有：

$$\int_A Z(\omega)\, d\mathbb{P}(\omega) = \int_A X(\omega)\, d\mathbb{P}(\omega)$$

现在取：

$$A=\Omega$$

因为：

$$\Omega\in\mathcal{G}$$

所以由条件期望的定义：

$$\int_\Omega Z(\omega)\, d\mathbb{P}(\omega) = \int_\Omega X(\omega)\, d\mathbb{P}(\omega)$$

也就是：

$$\mathbb{E}[Z]=\mathbb{E}[X]$$

又因为：

$$Z=\mathbb{E}[X\mid\mathcal{G}]$$

所以：

$$\boxed{ \mathbb{E}[\mathbb{E}[X\mid\mathcal{G}]] = \mathbb{E}[X] }$$

这就是塔式定理的基本形式。

 

## 一般形式

接下来证明塔式定理的一般形式。

设：

$$\mathcal{H}\subseteq\mathcal{G}$$

其中 $\mathcal{G}$ 表示更细的信息，$\mathcal{H}$ 表示更粗的信息。

要证明：

$$\boxed{ \mathbb{E}[\mathbb{E}[X\mid\mathcal{G}]\mid\mathcal{H}] = \mathbb{E}[X\mid\mathcal{H}] }$$

设：

$$Y=\mathbb{E}[X\mid\mathcal{G}]$$

再设：

$$U=\mathbb{E}[Y\mid\mathcal{H}] = \mathbb{E}[\mathbb{E}[X\mid\mathcal{G}]\mid\mathcal{H}]$$

我们要证明：

$$U=\mathbb{E}[X\mid\mathcal{H}]$$

根据条件期望的定义，只需要证明 $U$ 也满足 $\mathbb{E}[X\mid\mathcal{H}]$ 的两个条件。

 

### 1. 可测性

因为：

$$U=\mathbb{E}[Y\mid\mathcal{H}]$$

所以由条件期望的定义，$U$ 是 $\mathcal{H}$-可测的。

 

### 2. 部分平均性质

接下来证明，对任意：

$$A\in\mathcal{H}$$

都有：

$$\int_A U\, d\mathbb{P} = \int_A X\, d\mathbb{P}$$

因为：

$$U=\mathbb{E}[Y\mid\mathcal{H}]$$

所以对任意 $A\in\mathcal{H}$，都有：

$$\int_A U\, d\mathbb{P} = \int_A Y\, d\mathbb{P}$$

又因为：

$$Y=\mathbb{E}[X\mid\mathcal{G}]$$

并且：

$$A\in\mathcal{H},\qquad \mathcal{H}\subseteq\mathcal{G}$$

所以：

$$A\in\mathcal{G}$$

于是可以再次使用条件期望的定义：

$$\int_A Y\, d\mathbb{P} = \int_A X\, d\mathbb{P}$$

两步合起来：

$$\int_A U\, d\mathbb{P} = \int_A Y\, d\mathbb{P} = \int_A X\, d\mathbb{P}, \qquad \forall A\in\mathcal{H}$$

因此 $U$ 满足 $\mathbb{E}[X\mid\mathcal{H}]$ 的定义，所以：

$$U=\mathbb{E}[X\mid\mathcal{H}]$$

也就是：

$$\boxed{ \mathbb{E}[\mathbb{E}[X\mid\mathcal{G}]\mid\mathcal{H}] = \mathbb{E}[X\mid\mathcal{H}] }$$

这就是塔式定理。

 

## 直观理解

若：

$$\mathcal{H}\subseteq\mathcal{G}$$

则 $\mathcal{G}$ 是更细的信息，$\mathcal{H}$ 是更粗的信息。

其中：

$$\mathbb{E}[X\mid\mathcal{G}]$$

表示在细信息层面上对 $X$ 做平均。

而：

$$\mathbb{E}[\mathbb{E}[X\mid\mathcal{G}]\mid\mathcal{H}]$$

表示先在细信息块上求平均，再按照粗信息 $\mathcal{H}$ 合并平均。

塔式定理说明：

$$\boxed{ \text{先按细信息平均，再按粗信息平均} = \text{直接按粗信息平均} }$$

 

# 附录六、塔式定理的例子：三次不公平抛掷

设 $X_1,X_2,X_3$ 相互独立，且三次成功概率不同：

$$\mathbb{P}(X_1=1)=p_1,\qquad \mathbb{P}(X_2=1)=0.3,\qquad \mathbb{P}(X_3=1)=0.8$$

其中 $1$ 表示 $H$，$0$ 表示 $T$。

定义随机变量：

$$Y=X_1+X_2+X_3$$

即三次抛掷中 $H$ 的总个数。

定义两个信息层次。

细信息：

$$\mathcal{G}=\mathcal{F}_2=\sigma(X_1,X_2)$$

表示知道前两次结果。

粗信息：

$$\mathcal{H}=\mathcal{F}_1=\sigma(X_1)$$

表示只知道第一次结果。

显然：

$$\mathcal{H}\subseteq\mathcal{G}$$

 

## 第一步：按 $\mathcal{F}_2$ 平均

如果已经知道前两次结果，那么唯一不确定的就是第三次。

因为：

$$\mathbb{P}(X_3=1)=0.8,\qquad \mathbb{P}(X_3=0)=0.2$$

所以不能简单除以 $2$，而要按照 $0.8$ 和 $0.2$ 加权。

例如，前两次为 $HH$，也就是：

$$X_1=1,\qquad X_2=1$$

此时：

$$Y=2+X_3$$

所以：

$$\mathbb{E}[Y\mid X_1=1,X_2=1] = 2+0.8 = 2.8$$

也可以展开为：

$$3\cdot 0.8+2\cdot 0.2 = 2.4+0.4 = 2.8$$

注意这里不是：

$$\frac{3+2}{2}=2.5$$

因为第三次出现 $H$ 和 $T$ 的概率并不相同。

四个细信息块上的条件期望如下：

| 前两次结果 | $\mathbb{E}[Y\mid\mathcal{F}_2]$ |
|  -- |          --: |
| $HH$  |                    $2+0.8=2.8$ |
| $HT$  |                    $1+0.8=1.8$ |
| $TH$  |                    $1+0.8=1.8$ |
| $TT$  |                    $0+0.8=0.8$ |

也就是说：

$$\mathbb{E}[Y\mid\mathcal{F}_2]$$

在四个细信息块上的取值分别为：

$$2.8,\quad 1.8,\quad 1.8,\quad 0.8$$

 

## 第二步：再按 $\mathcal{F}_1$ 平均

现在只知道第一次结果。

### 第一次为 $H$

若：

$$X_1=1$$

则第二次仍不确定，并且：

$$\mathbb{P}(X_2=1)=0.3,\qquad \mathbb{P}(X_2=0)=0.7$$

所以粗平均不是：

$$\frac{2.8+1.8}{2}$$

而是：

$$0.3\cdot 2.8+0.7\cdot 1.8 = 0.84+1.26 = 2.1$$

因此，在 $X_1=1$ 时：

$$\mathbb{E}[\mathbb{E}[Y\mid\mathcal{F}_2]\mid\mathcal{F}_1] = 2.1$$

 

### 第一次为 $T$

若：

$$X_1=0$$

则第二次仍然满足：

$$\mathbb{P}(X_2=1)=0.3,\qquad \mathbb{P}(X_2=0)=0.7$$

所以：

$$\mathbb{E}[\mathbb{E}[Y\mid\mathcal{F}_2]\mid X_1=0] = 0.3\cdot 1.8+0.7\cdot 0.8$$

计算得到：

$$0.3\cdot 1.8+0.7\cdot 0.8 = 0.54+0.56 = 1.1$$

因此：

$$\mathbb{E}[\mathbb{E}[Y\mid\mathcal{F}_2]\mid\mathcal{F}_1] = \begin{cases} 2.1, & X_1=1,\\ 1.1, & X_1=0. \end{cases}$$

 

## 第三步：直接按 $\mathcal{F}_1$ 平均

现在直接计算：

$$\mathbb{E}[Y\mid\mathcal{F}_1]$$

若 $X_1=1$，则：

$$Y=1+X_2+X_3$$

所以：

$$\mathbb{E}[Y\mid X_1=1] = 1+\mathbb{E}[X_2]+\mathbb{E}[X_3] = 1+0.3+0.8 = 2.1$$

若 $X_1=0$，则：

$$Y=X_2+X_3$$

所以：

$$\mathbb{E}[Y\mid X_1=0] = 0+0.3+0.8 = 1.1$$

因此：

$$\mathbb{E}[Y\mid\mathcal{F}_1] = \begin{cases} 2.1, & X_1=1,\\ 1.1, & X_1=0. \end{cases}$$


## 结论

我们得到：

$$\mathbb{E}[\mathbb{E}[Y\mid\mathcal{F}_2]\mid\mathcal{F}_1] = \mathbb{E}[Y\mid\mathcal{F}_1]$$

这正是塔式定理：

$$\boxed{ \mathbb{E}[\mathbb{E}[Y\mid\mathcal{F}_2]\mid\mathcal{F}_1] = \mathbb{E}[Y\mid\mathcal{F}_1] }$$

它说明：先按更细的信息做条件期望，再按粗信息做条件期望，等于直接按粗信息做条件期望。


# 附录七、可测性

可测性在本文里主要承担一个直观角色：判断一个量是不是“只靠当前信息就能确定”。

设：

$$X:\Omega\to\mathbb{R}$$

是随机变量，$\mathcal{G}\subseteq\mathcal{F}$ 是一个子 $\sigma$-代数。

如果对任意 Borel 集合：

$$B\in\mathcal{B}(\mathbb{R})$$

都有：

$$\boxed{ \{\omega\in\Omega:X(\omega)\in B\}\in\mathcal{G} }$$

那么就说：

$$\boxed{ X\text{ 是 }\mathcal{G}\text{-可测的} }$$

其中：

$$\{\omega\in\Omega:X(\omega)\in B\}$$

是事件“$X$ 的取值落在 $B$ 中”。

直观地说：

> $X$ 是 $\mathcal{G}$-可测的，表示只根据 $\mathcal{G}$ 中的信息，就可以确定 $X$ 的取值。


## 有限样本空间中的理解

假设：

$$\Omega=\{HHH,HHT,HTH,HTT,THH,THT,TTH,TTT\}$$

表示抛掷三次硬币。

设：

$$\mathcal{G}=\mathcal{F}_2=\sigma(X_1,X_2)$$

表示只知道前两次抛掷结果。

那么 $\mathcal{G}$ 会将样本空间分成四个信息块：

$$A_{HH}=\{HHH,HHT\}$$

$$A_{HT}=\{HTH,HTT\}$$

$$A_{TH}=\{THH,THT\}$$

$$A_{TT}=\{TTH,TTT\}$$

定义：

$$Y(\omega)=\text{前两次中 }H\text{ 的个数}$$

那么：

$$Y(HHH)=Y(HHT)=2$$

$$Y(HTH)=Y(HTT)=1$$

$$Y(THH)=Y(THT)=1$$

$$Y(TTH)=Y(TTT)=0$$

可以看到，在 $\mathcal{G}$ 的每一个信息块中，$Y$ 的值都是常数。

所以：

$$Y\text{ 是 }\mathcal{G}\text{-可测的}$$

因为只知道前两次结果，就能确定 $Y$。


## 反例

如果定义：

$$Z(\omega)=\text{第三次是否为 }H$$

即第三次是 $H$ 时 $Z=1$，第三次是 $T$ 时 $Z=0$。

那么：

$$Z(HHH)=1,\qquad Z(HHT)=0$$

但是 $HHH$ 和 $HHT$ 在 $\mathcal{G}=\mathcal{F}_2$ 看来属于同一个信息块，因为前两次都是 $HH$。

如果 $Z$ 在同一个信息块中取不同的值，就说明它使用了第三次的信息。

因此：

$$Z\text{ 不是 }\mathcal{G}\text{-可测的}$$

在有限样本空间中，可以记住这个标准：

$$\boxed{ X\text{ 是 }\mathcal{G}\text{-可测} \Longleftrightarrow X\text{ 在 }\mathcal{G}\text{ 的每个信息块上都是常数} }$$

# 附录八、提取已知量

提取已知量就是前面递推中最常用的那一步：已经由过去信息决定的量，在对过去信息求条件期望时，可以像常数一样提出去。

条件期望有一个非常重要的性质，叫作“提取已知量”。

为了避免符号混乱，本节用 $Y$ 表示“已知量”，用 $X$ 表示“待平均的随机量”。

如果 $X$ 和 $XY$ 可积，并且 $Y$ 是 $\mathcal{G}$-可测的，那么：

$$\boxed{ \mathbb{E}[XY\mid\mathcal{G}] = Y\mathbb{E}[X\mid\mathcal{G}] }$$

它的意思是：

> 如果 $Y$ 的值已经能够由 $\mathcal{G}$ 中的信息确定，那么在对 $\mathcal{G}$ 求条件期望时，$Y$ 就可以像常数一样提出去。

如果 $X,Y$ 非负，则上式也成立，不过两边可能同时取 $+\infty$。


## 有限样本空间下的证明

设 $\mathcal{G}$ 是一个子 $\sigma$-代数。

在有限空间中，可以把 $\mathcal{G}$ 理解为一种“信息分辨能力”。它会把 $\Omega$ 分成若干个互不相交的信息块：

$$C_1,C_2,\ldots,C_m$$

满足：

$$\Omega=C_1\cup C_2\cup\cdots\cup C_m$$

并且：

$$C_i\cap C_j=\varnothing,\qquad i\ne j$$

这些 $C_j$ 就是 $\mathcal{G}$ 能分辨出的最小信息块。

例如，抛掷三次硬币，只知道前两次结果时：

$$\mathcal{G}=\mathcal{F}_2$$

此时信息块为：

$$C_1=\{HHH,HHT\}$$

$$C_2=\{HTH,HTT\}$$

$$C_3=\{THH,THT\}$$

$$C_4=\{TTH,TTT\}$$

我们证明：

$$\boxed{ \mathbb{E}[XY\mid\mathcal{G}] = Y\mathbb{E}[X\mid\mathcal{G}] }$$

只需要在每个信息块 $C_j$ 上证明它成立。

因为 $Y$ 是 $\mathcal{G}$-可测的，所以 $Y$ 在每个信息块 $C_j$ 上都是常数。

设：

$$Y(\omega)=y_j,\qquad \forall \omega\in C_j$$

也就是说，在信息块 $C_j$ 上，$Y$ 是常数 $y_j$。

以下默认 $\mathbb{P}(C_j)>0$。若 $\mathbb{P}(C_j)=0$，该块上的取值不影响条件期望的几乎处处意义。


### 1. 计算左边

在块 $C_j$ 上：

$$\mathbb{E}[XY\mid\mathcal{G}](\omega) = \mathbb{E}[XY\mid C_j], \qquad \omega\in C_j$$

展开为加权平均：

$$\mathbb{E}[XY\mid C_j] = \frac{ \sum_{\omega'\in C_j} X(\omega')Y(\omega')\mathbb{P}(\{\omega'\}) }{ \mathbb{P}(C_j) }$$

因为在 $C_j$ 上：

$$Y(\omega')=y_j$$

所以：

$$X(\omega')Y(\omega') = X(\omega')y_j$$

代入得：

$$\mathbb{E}[XY\mid C_j] = \frac{ \sum_{\omega'\in C_j} X(\omega')y_j\mathbb{P}(\{\omega'\}) }{ \mathbb{P}(C_j) }$$

由于 $y_j$ 对这个信息块里的所有 $\omega'$ 都相同，所以可以提出求和号：

$$\mathbb{E}[XY\mid C_j] = y_j \frac{ \sum_{\omega'\in C_j} X(\omega')\mathbb{P}(\{\omega'\}) }{ \mathbb{P}(C_j) }$$

而后面的这一项正是：

$$\mathbb{E}[X\mid C_j]$$

所以：

$$\mathbb{E}[XY\mid C_j] = y_j\mathbb{E}[X\mid C_j]$$


### 2. 计算右边

当：

$$\omega\in C_j$$

时，有：

$$Y(\omega)=y_j$$

并且根据有限样本空间中条件期望的分块定义：

$$\mathbb{E}[X\mid\mathcal{G}](\omega) = \mathbb{E}[X\mid C_j]$$

所以：

$$Y(\omega)\mathbb{E}[X\mid\mathcal{G}](\omega) = y_j\mathbb{E}[X\mid C_j]$$

这与左边完全相同。

因此，在每个信息块 $C_j$ 上，都有：

$$\mathbb{E}[XY\mid\mathcal{G}](\omega) = Y(\omega)\mathbb{E}[X\mid\mathcal{G}](\omega)$$

于是整个样本空间上成立：

$$\boxed{ \mathbb{E}[XY\mid\mathcal{G}] = Y\mathbb{E}[X\mid\mathcal{G}] }$$


## 核心理解

在每个信息块 $C_j$ 上，如果 $Y$ 是 $\mathcal{G}$-可测的，那么 $Y$ 就是一个固定常数 $y_j$。

所以：

$$XY=y_jX$$

于是：

$$\text{块内平均}(XY) = \text{块内平均}(y_jX) = y_j\text{块内平均}(X)$$

翻译成条件期望，就是：

$$\mathbb{E}[XY\mid\mathcal{G}] = Y\mathbb{E}[X\mid\mathcal{G}]$$

这就是“提取已知量”的本质。
